List of Topics:

  • Introduction to the Course, Text Processing: Basics, Spelling Correction: Edit Distance
  • Weighted Edit Distance, Other Variations, Noisy Channel Model for Spelling Correction, N-Gram Language Models, Evaluation of Language Models, Basic Smoothing
  • Language Modeling: Advanced Smoothing Models, Computational Morphology, Finite - State Methods for Morpholog
  • Introduction to POS Tagging, Hidden Markov Models for POS Tagging, Viterbi Decoding for HMM, Parameter Learning, Baum Welch Algorithm
  • Maximum Entropy Models - I, Maximum Entropy Models - II, Conditional Random Fields
  • Syntax - Introduction, Syntax - Parsing I, Syntax - CKY, PCFGs, PCFGs - Inside-Outside Probabilities, Inside-Outside Probabilities
  • Dependency Grammars and Parsing - Introduction, Transition Based Parsing : Formulation, Transition Based Parsing : Learnin
  • MST-Based Dependency Parsing, MST-Based Dependency Parsing : Learning
  • Distributional Semantics - Introduction, Distributional Models of Semantics, Distributional Semantics : Applications, Structured Models
  • Word Embeddings, Lexical Semantics, Lexical Semantics - Wordnet
  • Word Sense Disambiguation, Novel Word Sense detection, Topic Models : Introduction
  • Latent Dirichlet Allocation : Formulation, Gibbs Sampling for LDA, Applications, LDA Variants and Applications
  • Entity Linking, Information Extraction - Introduction, Relation Extraction, Distant Supervision
